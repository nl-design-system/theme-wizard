# Zoeken (Start) template

De startpagina voor de zoekfunctionaliteit. Deze template biedt een centrale plek waar bezoekers hun zoekopdracht kunnen starten.

## Opbouw van de pagina

- **Centrale zoekbalk**: Een prominente `h1` kop met een `SearchForm` om de bezoeker direct uit te nodigen tot interactie.
- **Paginastructuur**: Gebruikt de standaard `PageHeader` en `PageFooter` voor consistentie met de rest van de site.
- **Progressive Enhancement**: Werkt zowel client-side (met instant navigatie) als server-side (voor gebruikers zonder JS).

## Gebruik

Deze template is bedoeld als landingspagina voor de zoekfunctie, bijvoorbeeld bereikbaar via een "Zoeken" link in de header.
