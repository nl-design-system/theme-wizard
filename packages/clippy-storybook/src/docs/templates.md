# Templates en thema's

De Theme Wizard templates zijn voorbeeldpagina's die je kunt gebruiken als basis voor je eigen website. Elke template werkt met het **Start-thema** van NL Design System en past zich automatisch aan jouw huisstijl aan via design tokens.

## Hoe werkt het?

De visuele stijl (typografie, kleuren, ruimte) is vastgelegd in **basis-tokens** (`basis.*`). Door deze tokens aan te passen krijgt de template automatisch jouw huisstijl, zonder dat je de componenten zelf hoeft te wijzigen.

## Rebranden

Je kunt de stijl van een template aanpassen door:

- In de Theme Wizard de basis-tokens aan te passen (bijvoorbeeld `basis.text.*`, `basis.color.*`, `basis.space.*`)
- In Storybook een ander thema te kiezen via de themaselector om te zien hoe de template eruitziet met een andere huisstijl
